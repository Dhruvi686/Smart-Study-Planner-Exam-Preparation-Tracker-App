# Demo Flutter Folder Structure

This is a demo Flutter project structure organized for scalability.

## Structure

- `lib/app`: app-level setup (routes, themes, dependency setup)
- `lib/core`: reusable low-level code (constants, errors, utils)
- `lib/features`: feature-first modules
- `lib/shared`: shared widgets and helpers

## Run

1. Install Flutter SDK.
2. Open this folder in your IDE.
3. Run:

```bash
flutter pub get
flutter run
```
